# Eligibility

## Description
This application verifies if it is interesting for the customer and for the company to acquire the customer based on their consumption of electricity and returns from the sum of CO2 savings using clean energy.

## Build with
* Node 18 
* [NestJS](https://github.com/nestjs/nest) 
* Docker 
* Docker Compose
* Typescript

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
## Running the app with docker 
```bash
$ docker-compose up
```
## Installation

```bash
$ npm install
```

## Running the app
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
